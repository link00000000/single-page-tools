Use the player's own controls to play/scrub/pause. The buttons above just nudge by a second or an estimated frame for fine positioning. The file stays local — it's only read into the page.

2 — Capture a step

Pause the video where you want a step, then capture the frame. It becomes a new card on the right — add the instruction text there.

3 — Export whole guide

Copy for Confluence → copies formatted content (not raw tags) to your clipboard. Click into the Confluence Classic editor and paste — headings, images, and text land as real rich content. This is the one you want for Confluence.

Copy as Markdown → plain Markdown with embedded base64 images, useful for notes apps or Git-based docs.

Download .zip → a real .zip with numbered PNGs and a guide.md referencing them by filename, if you'd rather attach images to the page directly.
